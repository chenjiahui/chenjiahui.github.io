简单排序：

arr=[3,4,1]
temp = sorted(arr)
print arr
print temp

输出： 

[3, 4, 1]
[1, 3, 4]

简单排序示例2： 

arr=[3,4,1]
arr2 = arr.sort() 
print arr
print arr2

输出： 

[1, 3, 4]
None

正序排序： 

arr = [{'a': 3}, {'a': 4}, {'a': 1}]
sort=sorted(arr,key=lambda e:e.get('a'))
print sort

输出： 

[{'a': 1}, {'a': 3}, {'a': 4}]

倒序排序：

只需要设置reverse=True即可

arr = [{'a': 3}, {'a': 4}, {'a': 1}]
sort=sorted(arr,key=lambda e:e.get('a'),reverse=True)
print sort

输出： 

[{'a': 4}, {'a': 3}, {'a': 1}]
