    in other on 2014-04-26 | tags: java eclipse

    版权声明：本文为博主原创文章，未经博主允许不得转载。 https://blog.csdn.net/i_chenjiahui/article/details/24512669

    设置eclipse中的${user}

    eclipse中想要在创建java文件的时候显示@author "自己的名字"，

    在window -> preference -> java -> code typle -> code template -> Code -> New Java files ，点击Edit ，加入如下代码即可：

    /
     @author ${user} //创建者名字
     ${date} ${time} //创建时间

    /

    应用修改，这时，你会发现新建的java文件中@author 后面是administrator（windows系统的用户字）,解决方法如下。
