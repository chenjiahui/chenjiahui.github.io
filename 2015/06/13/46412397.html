Flex列排序

by Bug管理员
in 前端 on 2015-06-13 | tags: Flex

第一种：new一个sort，指定排序字段，然后赋给AdvancedDataGrid数据源的sort属性

var sort:Sort = new mx.collections.Sort();    
sort.fields = [new SortField("device_type",false,false),
new SortField("company",false,false),
new SortField("dist",false,false),
new SortField("device_name_sort",false,false)];

adv1.dataProvider.sort=sort;
adv1.dataProvider.refresh();

第二种：如果你的数据源是array，则使用array的sortOn方法，指定排序字段，如下：

(d.children as Array).sortOn("device_name_sort",Array.CASEINSENSITIVE);

第三种：点击列头排序，则需要自己指定列的排序方法： 设置col.sortCompareFunction=mySortCompareFunction;

private function mySortCompareFunction(obj1:Object, obj2:Object):int{
    if(obj1.sortname > obj2.sortname){
        return 1;
    }else if(obj1.sortname < obj2.sortname){
        return -1;
    }else{
        return 0;
    }
}
