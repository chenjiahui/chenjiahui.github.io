2、生成指定时间段的一个随机日期 

>>> from random import *
>>> from time import *
>>>
>>> date1 = (2015,1,1,0,0,0,-1,-1,-1)
>>> time1 = mktime(date1)
>>>
>>> date2 = (2016,1,1,0,0,0,-1,-1,-1)
>>> time2 = mktime(date2)
>>>
>>> random_time = uniform(time1,time2)
>>> print asctime(localtime(random_time))
Fri Sep 04 17:20:24 2015
>>>

3、随机生成一个100以下，3的倍数的数 

randrange ([start,] stop [,step])

start：开始的范围。这将包括在范围内。
stop：结束的范围。这将排除的范围。
step：添加一个数字来决定一个随机数的步骤。

>>> randrange(0,100,3)
15
>>> randrange(0,100,3)
18
>>> randrange(0,100,3)
45
>>> randrange(0,100,3)
72
>>> randrange(0,100,3)
12
>>> randrange(0,100,3)
75
