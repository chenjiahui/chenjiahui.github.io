java虚拟机-概述

by Bug管理员
in java on 2019-05-29 | tags: java

1、JVM JVM主要包含类加载器、执行引擎、内存区域、本地方法接口四个部分。

2、Java内存模型 内存区包含：方法区、虚拟机栈、本地方法栈、堆、程序计数器

方法区：存放类结构信息、常量池、静态变量、构造函数等 java堆：存储java实例的地方 虚拟机栈：java栈总是和线程关联再一起的，每创建一个线程，jvm就会为线程创建一个栈桢，用于存储局部变量、方法返回值、动态链接方法、操作数栈等。 本地方法栈：本地方法栈属于线程私有的数据区域，这部分主要与虚拟机用到的 Native 方法相关，一般情况下，我们无需关心此区域。 程序计数器：用来标示下一条指令 本地方法接口：主要是为调用c或c++实现的本地方法及放回结果。
